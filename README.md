# Projet CDA – DAB+

## Présentation

Ce projet est un mini-site web réalisé dans le cadre du cours de CDA.
Il simule un distributeur automatique de billets (DAB).

Le site contient :

- Une page d’accueil
- Une page ATM interactive (simulation de retrait avec billets et pièces)
- Un formulaire complet (plus de 15 champs différents)
- Un tableau de paramètres et d’historique
- Une galerie d’images (billets et pièces en euro)

---

## Comment utiliser le site ?

Aucune installation n’est nécessaire.

1. Télécharger le projet
2. Ouvrir le dossier
3. Double-cliquer sur le fichier `index.html`

Le site s’ouvre automatiquement dans le navigateur.

---

## Technologies utilisées

- HTML5 (structure)
- CSS3 (design et responsive)
- JavaScript (logique de la simulation ATM)

---

## Remarque importante

Les données saisies dans le formulaire ne sont pas enregistrées.
Il s’agit d’un projet pédagogique sans base de données.
